### 2.4 录音球 (Orb) 状态机

```
       点击 Orb
          │
    ┌─────┴─────┐
    │  idle     │──点击──→ listening (Recording audio…)
    │  finished │               │ 3s
    │           │               ↓
    └───────────┘          thinking (Processing…)
       ↑     ↑                 │ 1.8s
       │     │                 ↓
       │     └──── finished ── translating (Generating translation…)
       │                        │ 2.2s
       │ 点击停止               ↓
       │                   speaking (Playing back…)
       │                        │ 3.4s
       └────────────────── finished (Tap to record again)
```

**动画伴随**: idle → breathing glow | listening → pulse + ripple + 波形 | thinking → cyan spin | translating → 快速绿色脉冲 | speaking → 快速脉冲

### 2.5 逐句字幕时间线

| 时间 | 源语言显示 | 目标语言显示 |
|------|----------|------------|
| 3.0s | Hello everyone, | 大家好， |
| 4.2s | …welcome to the AI | …欢迎体验 AI |
| 5.4s | …real-time translation demo. | …实时翻译演示。 |

### 2.6 用户界面需对接的后端接口

```
POST /api/translate
  上传音频 + src_lang + tgt_lang
  Response: { taskId: 1, status: "processing" }

GET /api/translate/{id}
  轮询任务状态
  Response: { status: "completed", transcription: "...", translation: "...", segments: [...] }

GET /api/history?page=1&size=10
  获取当前用户翻译记录
  Response: { records: [...], total: 28, page: 1, size: 10 }

GET /api/history/{id}/export
  导出 TXT 文件

GET /api/auth/info
  Header: Authorization: Bearer <token>
  Response: { id: 1, username: "user", email: "...", role: "USER" }
```
