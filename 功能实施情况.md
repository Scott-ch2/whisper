### 2.4 录音球 (Orb) 状态机（已对接真实 API）

```
       点击 Orb / 拖拽文件
              │
        ┌─────┴─────┐
        │  idle     │──→ listening (MediaRecorder 真实录音 4s)
        │  finished │          │ 自动停止录音
        │           │          ↓
        └───────────┘     thinking (上传音频文件至 API)
           ↑     ↑              │ translateAudio()
           │     │              ↓
           │     └── finished ─ translating (轮询任务结果)
           │                       │ getTaskResult() 每 1.5s
           │                       ↓
           │                  speaking (播放完成)
           │                       │ 2s
           │                       ↓
           └───────────────── finished (Tap to record again)

也可通过 DragUpload 组件拖拽/选择音频文件直接上传翻译
```

**API 调用链路**:
1. `startRecording()` → `MediaRecorder` 录音 4s → 生成 `File(Blob)`
2. `translateAudio(audio, srcLang, tgtLang)` → `POST /api/translate` → 返回 `{ taskId, status }`
3. `processTask(taskId)` → `PUT /api/translate/{id}/process` → 触发后端处理
4. `getTaskResult(taskId)` 每 1.5s 轮询 → `GET /api/translate/{id}` → 返回 `{ status: "completed", transcription, translation, segments }`
5. 完成后显示双语卡片 + 逐句字幕时间线
